- **dir | dir ..\\address** 查看目录
- **cd** **| cd ..\\address** **| cd ..** **| cd \\** 换盘 | 切换路径  | 返回上一级 | 返回根目录
- **tree \\address** 查看目录的所有子目录
- **cls** 清屏
- **exit** 退出DOS
- **md | rd** 创建删除目录
- **echo XX > file** 创建文件
- **del file** 删除文件
- **copy file \\address\\file** 复制文件
- **move file \\address\\file** 剪切文件
