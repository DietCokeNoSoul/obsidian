==**java中允许有多个重名方法，但要求形参列表不一样。**==

减轻了给方法起名和记名的麻烦。

## 重载细节

- 方法名必须相同。
- 参数列表必须不同（参数类型、个数、顺序，参数名无要求）。
- 返回类型无要求。


---
## 可变参数

```java
class tools {
    public void sum(int... x) {
        for (int i = 0; i < x.length; ++i) {
            System.out.println(x[i]);
        }
    }
}
```


### >可变参数细节

- 可变参数的实参可以是0个或者任意多个。
- 可变参数的实参可以为数组。
- 可变参数的实质就是数组。
- 可变参数可以和普通参数一起放在形参列表，但必须保证可变参数在最后。
- 一个形参列表中只可能出现一个可变参数。
